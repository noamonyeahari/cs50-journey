## Key Concepts Learned

### 1. Computational Thinking

Computational thinking is a structured approach to solving problems.

It involves:

#### Decomposition

Breaking a large problem into smaller manageable parts.

Example:

Selling land can be broken into:

* Lead generation
* Qualification
* Property presentation
* Site inspection
* Negotiation
* Documentation
* Payment
* Handover

---

#### Pattern Recognition

Identifying repeated activities or processes.

Example:

Most property transactions follow similar steps.

---

#### Abstraction

Focusing on important information while ignoring unnecessary details.

Example:

For a property inquiry, the important information includes:

* Budget
* Location
* Purpose
* Plot size

---

#### Algorithm Design

Creating a sequence of steps to solve a problem.

---

### 2. Algorithms

An algorithm is a set of instructions used to solve a problem.

Key lesson:

Programming starts with designing the solution before writing code.

---

### 3. Decision Making

Many algorithms contain decision points.

Example:

Prospect ready to proceed?

YES → Continue transaction

NO → Schedule follow-up

This resembles how computers use conditions and logic.
